# 📱 Flutter Instagram Clone

## Overview

This project is a Flutter-based Instagram-inspired mobile application developed as part of my learning journey in mobile application development. The application focuses on creating a responsive user interface and implementing CRUD operations for managing user content.

---

## Features

- User-friendly interface inspired by Instagram
- CRUD functionality
- Responsive mobile UI
- Navigation between screens
- Flutter widgets and layouts
- State management

---

## Technologies Used

- Flutter
- Dart
- Android Studio
- VS Code

## How to Run

1. Clone the repository

```bash
git clone https://github.com/sanaasaleem76/flutter-instagram-clone.git
```

2. Install dependencies

```bash
flutter pub get
```

3. Run the application

```bash
flutter run
```
